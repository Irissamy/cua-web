天堂遊戲｜金富發

天堂遊戲2016年在台北松菸文創設立據點。
用熱愛激發每個人的潛能，不懈追求，以平凡創造不凡。
用想像釋放每個人的夢想，不斷進取，以創意創造價值。
對員工和專業人才，尊重個性，平等對待，提供良好的工作環境與發展空間。
扁平化的組織架構，順暢的溝通機制；簡單純粹的做事理念，創造高品質產品。

學習

創新求變不斷進步，我們鼓勵大家學習。到處學習尋找靈感不吝分享。

分享

定期舉辦技術交流，持續進行知識分享，技術端互相交流集思廣益。

創新

與客戶合作建立友善的產品，實踐新的想法並開發更好的功能。

我們的團隊

天堂是由知名遊戲、網路公司核心人才和技術菁英帶著對夢想的追求所組成。
規模仍在與日俱增，為了不同領域的挑戰與解決方案，我們正在招賢納士。
如果您喜歡克服難題，並且熟悉代碼編寫，那你就是我們所需要的人！

Frontend

前端團隊由工程師(WEB & APP)與設計師組成，秉持自由、多元、開放，沒有繁瑣的規定，只有共同團隊目標，讓夥伴適性發展。
成員中有業界資深熱血夥伴，加速接觸各專業領域的知識，一同激盪出技術的火花。

身為一個天堂遊戲的前端工程師，你將會:
1. 與設計師和後端工程師合作，設計前端介面與後端API串接
2. 撰寫出高復用性元件以及兼具可讀性與模組化的程式代碼
3. 評估並運用新技術，優化工作流程或開發模式，確保程式品質、執行效率及可維護性
要哪些技術呢?
1. 熟悉 Html, CSS, JavaScript 等網頁相關基本技術
2. 基本 Git 能力及熟悉任何一套 javaScript framework，或熟悉 ReactNative

身為一個天堂遊戲的前端設計師，你將會:
網站與APP的介面設計與互動規劃、專案執行能力與開發、技術、軟體、設計趨勢的學習與分享
要哪些條件呢?
1. 良好的色彩感、瞭解網頁設計與平面設計的差別
2. 熟悉頁面切版和CSS、HTML5等

DevOps

我們是一群由QA、IT、RD組成的團隊，平均年紀27歲是一個相當年輕的團隊，主要的任務是
1. 開發遊戲渠系統，每月處裡千萬訂單的流量
2. 協助公司產品 (Web,Mobile Web,APP)測試
3. 建立監控機制，處理 CI/CD 流程及雲端的管理

我們團隊有用到哪些技術呢?
1. Golang/C#/Python
2. Jenkins/Gitlab/Slack/Redmine/TestLink
3. Selenium/Docker/AWS/GCP/Azure
4. Mysql/Elasticsearch
5. Prometheus/Graylog

Backend

我們是一個樂於學習分享的團隊，我們不怕失敗，只怕不敢失敗。
我們提供可接受失敗的環境，從失敗中成長
當然不是允許開發出嚴重的bug，而是我們願意一起承擔我們所能承擔的風險。

《需求》
1.開發月交易量千萬以上的交易平台，參與開發之分析/設計/實現。平台包含了交易系統、API服務、報表系統等
2.熱愛團隊合作。
3.對程式開發工作有熱情者或對技術整合有深度渴望
4.歡迎想挑戰自己的撰寫的程式能服務千萬交易量服務的工程師應徵

《挑戰》
1. 系統架構：需負責系統之分析、設計、程式撰寫與單元測試，含後端系統架構與資料管理，並提供RESTful API與其他系統(如mobile Apps)串接
2. 海量資料：巨量資料的處理與分析
3. 7天x24小時營運：保持持續的系統不中斷營運
