實習生計畫 熱烈展開

在成為職場新鮮人之前，聚賢科技提供實習機會，讓你/妳在畢業之前來體驗職場。
每年5-7月及11-12月履歷篩選、面談，限大三生大四及研究生。
透過實習讓你贏在起跑點!快來加入我們吧~

招募流程

1.履歷資料篩選

2.筆試

3.面談

4.錄取

5.報到實習

招募職種介紹
實習職務 開發/QA工程師(golang) 前端工程師
內容 1.協助參與公司專案的開發，編寫指定的模組功能
2.通過 Web 前/後端開發，帶給用戶更好的使用體驗
3.產品測試 ( WEB, Mobile Web, Native APP, 桌面應用) 		1. 網站專案的前端建置、功能開發
2. 與設計師和後端工程師合作，設計前端介面與後端API串接
3. 撰寫出高復用性元件以及兼具可讀性與模組化的程式代碼
教學方式 專人帶領→設定學習目標→學習檢視
實習規定 半年為一期；週一~週五中至少可上班3天
條件 1.至少對其中一門編程語言(C#, Java or Python等)
2.積極主動，有上進心，願意學習新知識 		1. 熟悉 Html, CSS, JavaScript 等網頁相關基本技術
2. 至少使用過一套前端框架 ( Vue 或 React 或 Angular)
3. 積極主動、對前端開發有熱忱
其他 先至QA單位學習、了解產品後→依照個人能力及興趣分配到各組別
